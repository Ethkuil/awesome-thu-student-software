# awesome-thu-student-software
本项目收录主要由清华大学在校学生开发/维护的实用开源软件。目前收录的软件都聚焦于校园生活。

要求：
* 实用、可用。具有重要纪念/参考意义的可破例，如T大树洞。
* 「仅供学习参考、不应传播」的软件一般不收录，如文泉学堂PDF下载。
